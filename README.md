# Bootstrap base
Use it like a template that starts production with Bootstrap.
Creating a theme's CSS uses Customizer.
For fine rewriting, use style.css.
## Included components
- Header menu
- Responsive content
  - Fixed
  - Fluid
- Modal window
- Slider
